What is KS Photography?

General

KS Photography is a dual-purpose web application designed to demonstrate my abilities as a software engineer while showcasing my own photography.

Limited Functionality

As a portfolio example, I believe it is important to present a fully functional application. However, I feel it necessary to employ certain restrictions in order to avoid possible misrepresentation of myself. Therefore, all user-initiated additions, edits and updates have been purposely limited and use front-end storage only. This means that no user changes will permanently impact the content of the application.

Construction

This application utilizes the MEAN stack of web development. It is written in JavaScript using AngularJS and Node.js, employs a MongoDB database, and is stylized with Google Fonts, Bootstrap and UI Bootstrap, and Masonry. All photos are original images taken by and/or owned by Kyle Seem.

You can review the code for this application in my GitHub repository. Thank you for visiting!
