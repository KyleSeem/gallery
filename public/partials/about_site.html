What is KS Photography?

General

KS Photography is a web app that lets me manage and share my photography in a gallery format. The primary purpose is to demonstrate my abilities as a web developer/software engineer.

Construction

I built this application using the MEAN stack of web development. It is written in JavaScript using AngularJS and Node.js, employs a MongoDB database, and is stylized with Bootstrap and UI Bootstrap, Google Fonts, and Masonry. All photos and backgrounds are original images created and/or owned by Kyle Seem.

The code for this application can be viewed in my GitHub repository.

Limited Functionality

As this is a portfolio example, I want to show that this is not just a gallery, but an interactive application with a UI that allows me to fully manage my content. For me, part of designing a good application is making it easy to use and low-maintenance. I want you to be able to experience the app like I do, so the update > add new photo section has a group of images that can be added to, edited and deleted from the gallery. Any changes you make are temporary and can only be seen on the device you are using. So go nuts!

Wait... if it's all temporary, why can't you just upload your own images? The short answer is lack of storage. Keeping your changes local means other users will see the original gallery when they visit the site. It also means I don't have to worry about what content shows up in my gallery. Local storage is limited and storing more than just some settings data would significantly slow things down or even just not work. I'm still trying out different options in my spare time so it may be available in the future.

Cookies

Cookies are used to identify if a user has visited or made any changes to the site. I collect zero personal information and all data is automatically cleared after 24 hours.
